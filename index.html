01 About Me

Pursuing Computer Science at UWaterloo & Business Administration at WLU

Hey! I'm abdullah khan, and I'm doing something a bit unconventional—I'm pursuing a double degree in Computer Science at the University of Waterloo and Business Administration at Wilfrid Laurier University. The double degree keeps me busy, but it also gives me a unique perspective on how technology and business intersect—and that's exactly where I want to be.

03 Featured Work

Projects

Security Tool

Attack Surface Growth Simulator (ASGS)

Built a comprehensive full-stack security risk modeling tool that calculates attack surface scores (0-100) across 5 threat categories by normalizing system metrics including endpoints, users, MFA adoption, and vulnerabilities. The system generates ranked driver breakdowns with actionable security recommendations.

  • Developed FastAPI backend with Pydantic validation that computes quadratic risk functions and calculates first/second derivatives using NumPy to identify unsafe growth zones where system complexity creates disproportionate security exposure
  • Designed SQLite database with SQLAlchemy ORM to persist assessment configurations and results, enabling users to compare different security scenarios and track how architectural changes impact overall risk posture
  • Built interactive React dashboard with Recharts that visualizes risk curves, growth rates, and danger zones in real-time as users adjust system parameters, helping teams identify critical inflection points before scaling
Python React FastAPI SQLAlchemy NumPy SQLite Recharts Pydantic
Automation Tool • Hackathon

AutoForm (Spur Hackathon)

Built a PDF auto-fill application that extracts user profile data and programmatically populates common forms using OpenRouter API for intelligent field mapping and pdf-lib for PDF manipulation. Developed during a hackathon to automate tedious form-filling processes.
